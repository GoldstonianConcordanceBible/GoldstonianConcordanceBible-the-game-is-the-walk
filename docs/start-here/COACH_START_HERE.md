# Coach Start Here

This repo supports the book method with **printable tools**.

## Best way to use (Team)
Weekly rhythm:
- Day 1: Scripture Rep + Drill (10 min)
- Day 2: Film Room reflection (5–10 min)
- Day 3: Game Day Walk challenge (2 min)
- End of week: Culture standard reminder (2 min)

## Print these
- Practice Cards: `tools/practice-cards/`
- Weekly Plan template: `tools/weekly-plans/WEEKLY_PLAN_TEMPLATE.md`
- Team Standards: `tools/standards/TEAM_STANDARDS_TEMPLATE.md`

## Coach script (30 seconds)
“The game is the Walk. We train discipline, truth, humility, and consistency. Reps make us ready.”
