# Study Guides — The Game is the Walk

These guides are **chapter-aligned**, designed for:
- Teams (10–20 minutes)
- Families (7–12 minutes)
- Small groups (30–45 minutes)

Each guide includes:
- Rep (Scripture references only)
- One Big Idea
- Discussion
- Drill (one action)
- Film Room
- Closing prayer (one sentence)

Note: These are support materials, not the full book text.
