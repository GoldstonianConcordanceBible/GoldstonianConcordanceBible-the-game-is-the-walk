# The Game is the Walk: Practice Like a Pro in God’s Word
A Goldstonian Concordance Bible (GCB) Applied Studies project.

This repository is the **living training system** supporting the book:
**The Game is the Walk: Practice Like a Pro in God’s Word**.

The book is the **message + method**.
This repo is the **reps + tools + coach kits + printable practice cards**.

## What this repo is
- A **coach-ready** and **family-ready** discipleship playbook
- Printable **Practice Cards**, trackers, and weekly plans
- A structured **“film room”** system for reflection + improvement
- “Team Culture” standards templates for programs and homes
- **Divine Agents** (safe, structured AI prompts) to help generate study guides, lesson plans, and devotion variants

## What this repo is NOT
- Not the full book text (copyright stays protected)
- Not a social feed
- Not random notes
